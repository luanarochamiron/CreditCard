# Credit Card

Uma interface de cartão de crédito moderna e elegante desenvolvida com HTML5 e CSS3. O projeto destaca a aplicação do efeito Glassmorphism (vidro fosco), utilizando propriedades avançadas de estilização para criar profundidade e realismo visual.

## Tecnologias Utilizadas

* HTML5 (Estruturação semântica)
* CSS3 (Estilização avançada e posicionamento)
* Efeito Glassmorphism (`backdrop-filter: blur`)
* Flexbox / CSS Grid (Alinhamento dos elementos)

##  Funcionalidades e Técnicas Aplicadas

* **Efeito de Vidro Fosco:** Uso de transparências com cores RGBA combinadas com desfoque de fundo.
* **Elementos Flutuantes:** Formas abstratas posicionadas estrategicamente atrás do cartão para evidenciar o efeito translúcido.
* **Design Premium:** Detalhes simulando um cartão real, como o chip dourado e a disposição das informações textuais (número, nome e validade).
